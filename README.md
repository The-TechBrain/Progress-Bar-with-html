# Glowing Progress Bar

This is a glowing animated progress bar built using HTML, CSS, and JavaScript.  
It auto-fills from 1 to 100% and is perfect for modern UI dashboards or loaders.

## How to Use
1. Clone the repo or copy code
2. Open `index.html` in browser
3. Customize styles or progress logic in JS
